### ⚠️一定要区分this.$route 和 this.$router 这两个对象

1.this.$route 是路由参数对象，所有路由中的参数，params 和 query 都属于他
2.this.$router 是一个路由导航对象，用它可以方便的使用js代码，实现路由的前进 后退 跳转到新的URL地址
